📸 如何添加您的背景图片

  1. 将您的背景图片命名为 background.jpg
  2. 将图片放置在 assets/images/ 目录中
  3. 刷新页面即可看到效果

🔧 CSS代码示例

以下是添加背景图片的CSS代码：

body::before { content: ''; position: fixed; top: 0; left: 0; width: 100%; height: 100%; background-image: url('/assets/images/background.jpg'); background-size: cover; background-position: center; background-repeat: no-repeat; background-attachment: fixed; opacity: 0.3; /* 调整透明度 */ z-index: -2; pointer-events: none; }

💡 使用建议

  • 图片尺寸: 建议使用 1920x1080 或更高分辨率
  • 文件格式: JPG格式最佳，文件大小建议小于2MB
  • 透明度: 30%透明度通常最适合，不会影响文字阅读
  • 响应式: 背景图片会自动适应不同屏幕尺寸

🎯 当前演示说明

此页面使用了一个SVG渐变背景作为演示。在实际使用中，您需要将 background-image 替换为您的图片路径。
